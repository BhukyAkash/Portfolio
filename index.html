PROJECTS

AWS 3-TIER WEB APPLICATION

The project implements a 3-tier architecture, dividing the application into three logical layers: Web Tier, Application Tier, and Database Tier. The frontend is hosted on Amazon S3, delivering static files globally with high reliability, while Amazon Route 53 manages DNS and domain mapping for seamless accessibility. The application layer is powered by Amazon EC2 instances, orchestrated with Elastic Load Balancer (ELB) and an Auto Scaling Group for dynamic resource scaling. The database layer employs Amazon RDS for managing relational data, featuring Multi-AZ deployment for high availability and automatic backups for data integrity.
